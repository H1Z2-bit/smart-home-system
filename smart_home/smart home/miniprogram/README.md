# 智能家居综合管理系统小程序端

本目录是基于 uni-app + Vue3 的微信小程序骨架，已预留对接当前 FastAPI 后端的接口。

## 使用方式

1. 用 HBuilderX 打开本目录：

```text
C:\Users\hanzhe\Desktop\software\smart home\miniprogram
```

2. 修改后端地址：

```text
config/index.js
```

将 `BASE_URL` 改成你的电脑局域网 IP，例如：

```js
export const BASE_URL = 'http://192.168.1.23:8000'
```

3. 启动后端：

```powershell
cd "C:\Users\hanzhe\Desktop\software\smart home\backend"
conda activate visgeom
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

4. 微信开发者工具中开启：

```text
详情 -> 本地设置 -> 不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
```

5. HBuilderX 中运行：

```text
运行 -> 运行到小程序模拟器 -> 微信开发者工具
```

## 已完成页面

- 登录页：账号密码登录、Mock 手机验证码登录
- 首页：用户信息、家庭空间、设备/报警状态总览、快捷入口
- 房间管理页：房间列表、新增房间、编辑房间、删除房间
- 设备页：房间筛选、设备列表、开关控制、新增设备、编辑设备、删除设备、设备详情入口
- 设备详情页：设备状态、关键设备标记、快速控制、模拟指标
- 设备模拟与自检：提交模拟数据、查看模拟记录、发起自检、查看自检记录
- 场景模式页：查询场景、一键执行场景
- 定时任务页：新增、编辑、删除、启停定时任务
- 联动规则页：新增、编辑、删除、启停联动规则，支持 JSON 条件与动作配置
- 空间可视化页：2.5D 房屋视图、房间设备分布、安全设备标记
- 系统配置页：报警阈值、自动报警开关、模拟开关
- 系统日志页：操作日志、系统日志
- 演示模式页：Mock 数据重置、答辩演示流程
- 报警页：报警统计、报警列表、确认、处理、关闭、误报
- 个人中心：资料查看、绑定手机号、系统入口、后端地址配置、连接测试、退出登录
