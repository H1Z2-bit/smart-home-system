# 智能家居 3D 可视化 Demo

这是独立于正式业务前端页面的 Three.js 可视化演示页，用于展示智能家居系统的房间布局、设备状态、传感器数据和报警联动效果。

## 启动

```powershell
cd C:\Users\hanzhe\Desktop\software\frontend
npm.cmd install
npm.cmd run dev
```

浏览器访问终端显示的地址，通常是：

```text
http://localhost:5173/
```

## 当前完成内容

1. 四个房间：客厅、厨房、卧室、玄关。
2. 细化户型：地板、墙体、门洞、窗户、入户门、房间标签。
3. 家具细节：沙发、茶几、电视柜、床、枕头、衣柜、橱柜、水槽、冰箱、鞋柜等。
4. 设备模型：智能灯、智能插座、摄像头、温度传感器、烟雾传感器、燃气传感器、门磁传感器。
5. 状态表现：在线、离线、开关状态、传感器读数、报警状态。
6. 报警效果：厨房区域红色高亮、报警设备脉冲、报警提示条。
7. 交互能力：点击 3D 设备查看详情、房间筛选、设备列表选择、总览/报警定位/设备跟随视角切换。
8. 后端映射：右侧面板显示对应接口路径，便于后续接入真实后端 API。
9. 响应式布局：支持桌面和移动端展示。

## 后续接入方向

1. 将当前静态设备数组改为从 `/api/homes/1/devices` 拉取。
2. 将传感器读数接入 `/api/devices/{device_id}/simulations`。
3. 将报警效果接入 `/api/alarms` 和 `/api/devices/{device_id}/simulate`。
4. 将设备控制按钮接入 `/api/devices/{device_id}/control`。
5. 接入正式 Vue 路由和菜单，成为系统首页或设备可视化页面。
